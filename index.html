Javascript (JS)

is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

are used to assign values, compare values, perform arithmetic operations, and more.

Variables

JavaScript variables are containers for storing data values.

Variable Naming conventions

All names start with a letter. Use camelCase for identifier names. Variables should be self explanitory.

Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure.

Methods

Methods are the actions that can be performed on objects.

Arguments

An argument is the value assigned to a parameter.

Parameters

A variable assigned to an argument.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

Chaining Methods, also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code.

Strings

Strings are a datatype that represents the exact text of whatever is enclosed in quotes.

Booleans

The boolean is an object that represents two values, either "true" or "false".

Undefined

Undefined means a variable has been declared, but the value of that variable has not yet been defined.

NaN

An integer which is also not a number.

Escape

The escape function encodes a string.

Alert

the Alert method displays an alert box with a specified message and an OK button.

Comments

Text used to explain the code.

JQuery

Is a framework to help make writing javascript easier.

Attribute

Provides additional information about html elements.
